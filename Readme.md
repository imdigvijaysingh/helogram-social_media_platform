# 🌸 OnBoard - Social Media App

OnBoard is a modern social media platform built using the MERN stack.  
It allows users to create accounts, share posts, and interact with other users in a simple and engaging way.

---

## 🚀 Features

- User Authentication (Signup / Login)
- Secure password hashing
- Set up profile (Profile Photo, Username and date of birth)
- Create and upload posts with image and caption
- Image upload support
- Modern UI with clean error handling
- Responsive design

---

## 🛠 Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** bcrypt
- **Storage:** Cloud storage for images - Imagekit.io

---

## 📂 Project Status

🚧 Currently in development.

Authentication system and core frontend/backend integration are completed.  
Post creation, profile features, and additional social interactions are under development.
